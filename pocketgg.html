pocket.gg Privacy Policy


smash.gg API & Auth Token

pocket.gg uses smash.gg's GraphQL API to retrieve data, such as tournament details. The API requires an auth (authentication) token for access, which can be obtained for free using a smash.gg account. Please visit this page for instructions on how to obtain a smash.gg auth token.

pocket.gg does not ask for or collect any personally identifiable information; only a smash.gg auth token. By providing pocket.gg with an auth token, you are allowing the app to make requests using the API and retrieve data from smash.gg, without having to provide pocket.gg with your smash.gg password. The auth token is stored locally, and is never sent to an external server (besides being used for authentication for smash.gg's servers).

The auth token does not provide pocket.gg access to your smash.gg password, and any details about the user that are displayed in the app (e.g. the profile screen displays the user's gamertag and tournaments) are never sent to an external server.


Crash Reporting & Analytics

pocket.gg uses Firebase for crash reporting and analytics, to improve and maintain the stability of the app. This automatically reports any errors that may occur (such as app crashes or errors when generating a bracket view), which greatly expedites the ability to fix them.
